This screen — Post dashboard (/admin)

  • The author's home for managing all posts, drafts included (§4.1).
  • The toolbar has the title and quick links (View site · Series · Comments · New post · Log out).
  • Each row has a status label (Published/Draft), category, publish date, title, and Edit · Delete actions — plus View only for published posts (a draft has no public page).
  • Delete opens a confirm dialog first — it never deletes on the first click (§7.5 Confirm dialog).
  • When there are none: "No posts yet."
